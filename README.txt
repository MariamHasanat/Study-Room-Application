# Study Room Application

## الميزة المنفذة
تم إضافة ميزة تسجيل الدخول باستخدام حساب Google عبر Firebase Authentication. عند تسجيل الدخول لأول مرة بحساب Google، يتم إنشاء مستخدم جديد في قاعدة بيانات Firestore تلقائيًا.

## طريقة تشغيل المشروع

1. **تأكد من وجود Node.js أو استخدم Live Server في VS Code.**
2. شغّل المشروع محليًا (مثلاً عبر Live Server أو أي سيرفر محلي).
3. افتح الرابط:  
   ```
   http://127.0.0.1:5500/pages/login.html
   ```
